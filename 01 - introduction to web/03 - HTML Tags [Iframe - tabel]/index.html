table tag
for creating table in html we have 3 tags should be created
table: for creating main layout for the table
tr: for creating layout for row in the table
td: for creating layout for column in the table
also we having more tags used with table like
th: for creating layout for making cell as header of column
caption: this's tag are used for adding caption for the table,
thead: for creating layout for table heading
tbody: for creating layout for table body
tfoot: for creating layout for table footer.
also the table having an attributes like [border - rowspan ( adding to td )- colspan ( adding to td)] but border tag is deprecated now and the common way to adding border we will use css border.
- we used table for one of two reassons
  1. for creaing table
  2. for making page layout (but it's old way and deprecated in html 5)
