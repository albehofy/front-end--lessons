selectors type
we have many type of selectors in css
  1. universal selector *: used for selecting all (select every thing inside the page)
  2. type selector tag-name: used for selecting tag.
  3. attribute selector
    elemnt [ attribute || attribute='value' (for getting a specefic attribute)]: used for selecting attribute
    element [attribute ~="value"] : this means if the attribute containing value.
    element [attribute ^="value"] : this means if the attribute start with value.
    element [attribute $="value"] : this means if the attribute end with value.
  4. ID selector #id-value: used for selecting element wich including selected ID.
  5. > selector element > child: used for selecting direct child of the element.
  6. space selector element child: used for selecting child of the element.
  7. + selector element + sibling: used for selecting direct sibling of the element.
  8. ~ selector element ~ sibling: used for selecting sibling elements of the element.
